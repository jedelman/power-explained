continuous arc the mechanical split fragmented. Section-separated pairs are
excluded — the `---` is authorial intent to hold apart.

Generated by `scripts/merge-map.mjs` → `gen-merge-map-doc.mjs`. Regenerate:
`node scripts/merge-map.mjs --json && node scripts/gen-merge-map-doc.mjs`.

**Bands:** `STRONG` (score ≥ 3, high confidence) · `consider` (1–2) · `keep` (≤ 0).
Take STRONG with light review; weigh `consider` case by case; `keep` left out.

**To merge a pair** `G-XX-AAA → G-XX-BBB`: append B's body to A's (blank line
between), delete B's file, remove B's id and the paragraph separator at that
index from the plateau manifest. A's id is kept; B's id is retired. Re-run
`compile-plateau verify` against the snapshot — a merge is byte-preserving
(it only removes a `\n\n` join, which the composer re-adds), so the snapshot
check still passes.
