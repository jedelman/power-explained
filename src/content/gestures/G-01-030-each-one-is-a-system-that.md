  Each one is a system. Each one takes something away from the community. It
  takes away the way people move around, or their homes, or their say in
  decisions. In return it gives back something else. More traffic moving
  through. Bigger institutions. The look of a fair process. In each one,
  information comes up from the people below. But it does not change the
  outcome. And the people running the system are mostly not cruel. They are
  inside a pattern that feels so normal that it works on its own. No one has to
  force it.

Each one is a system that extracts something from the community — mobility, housing, political agency — and returns something else — throughput, institutional development, the appearance of process. In each one, information flows in from below and doesn't change the outcome. In each one, the people operating the system are not, mostly, malicious. They are inside a template so naturalized it functions without anyone having to enforce it.
